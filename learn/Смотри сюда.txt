Ключевые слова
  01) False - Ложь
  02) True - Правда
  03) None - пустой объект
  04) and - логическое И
  05) with/as - менеджер контекста
  06) assert - условие - возбуждает исключение, если условие ложно
  07) break - выход из цикла
  08) class - пользовательский тип данных состоящий из методов и атрибутов
  09) continue - переход на следующую иттерацию цикла
  10) def - определение функции
  11) del - удаление объекта
  12) elif - в противном случае, если
  13) else - if/else (если/то)
  14) except - перехватить исключение
  15) finally - в купе с инструкцией try, выполняет инструкцию независимо от того, было исключение или нет
Типы данных
  1) None - неопределённое значение переменной
  2) Boolean type - Логические переменные 
    1) True - истина
    2) False - Ложь
  3) Numeric type - Числа
    1) int - целые числа
    2) float - число с плавающей точкой
    3) complex - комплексное число
  4) Secuence type - Списки
    1) list - список
    2) tuple - кортеж
    3) range - диапазон
  5) Text secuence type - Строки
    1) str - строка
  6) Binary secuence types - Бинарные Списки
    1) bytes - байты
    2) bytearray - массивы байт
    3) memoryview - специальные объекты для доступа protocol buffer
  7) Set types - Множества
    1) set - множество
    2) frozenset - неизменяемое множество
  8) Mapping types - Словари
    1) dict - словарь
